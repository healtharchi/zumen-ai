🧪 これは「スマート見積＋図面AI」の実験版です

いつものスマート見積システムに、「図面を放り込むと、外周長・床面積などの入力欄にAIが自動で数字を入れてくれる」機能を実験的に足したものです。本番の公開ツールとは別物（試作）で、精度を確かめる段階です。

使う前の準備（最初に1回だけ）

  1. 見積を開いて「1. 規模」タブを表示します。
  2. 一番上の 🧪 図面AIで自動入力 パネルの中の「🔑 Claude APIキー」を開きます。
  3. お持ちの Claude API キー（sk-ant-…）を貼り付け、「🔒 キーを保存」を押します。
  4. キーはこの端末のブラウザの中だけに保存されます（外部サーバーには送りません。送り先はClaude公式のみ）。

図面から積算する流れ

  1. 「1. 規模」タブの 🧪 図面AIで積算 パネルに、図面をドラッグ＆ドロップします。平面図・立面図・パースをまとめて複数枚入れるほど精度が上がります（PDF・JPG・PNG）。
  2. 縮尺の手がかり（任意）に、わかっている寸法や面積表の値を入れると精度が上がります（例：「南面の通り芯が7280mm」「面積表に延床104㎡」）。
  3. オレンジの 「🔍 図面をAIで積算して入力欄に反映」 を押します（図面の枚数で30秒〜2分）。
  4. AIが読み取った値が チェックリストで出ます。信頼度が高いものは最初からチェック済み（＝自動入力）。クロス・幅木など低めの項目も、チェックを入れればその場で反映できます。
  5. 採用した値が、外周長・外壁面積などの「規模」欄、クロス・フローリング・幅木・廻り縁などの「2. 内訳明細」の数量に入り、見積金額もその場で更新されます。
  6. 入った数字は各入力欄でいつでも手直しできます（AIはあくまで下書き）。

仕組みのポイント

  • AIは図面ファースト（図面の寸法・記号を最優先）で、読めない箇所も910グリッド等の標準仮定で必ず数値を出します。各値に実測／推定と信頼度が付きます。
  • チェックを外せばその項目は従来の係数推定に戻ります（破綻しません）。
  • 仕上表・建具表が無いと、クロスや各室仕上げは推定（信頼度低め）になります。図面が多いほど精度が上がります。

うまくいかないとき

  • 「APIキーが正しくないようです(401)」→ 🔑からキーを入れ直してください。
  • クロスなどが入らない→ リストに出た項目のチェックを入れれば反映されます（信頼度が低めでも採用できます）。
  • 精度を上げたい→ 平面図に加えて立面図・パースも一緒に入れる／縮尺の手がかり（既知寸法・面積表）を入れてください。
  • 図面がぼやけている→ できるだけ鮮明な画像／PDFを使うと精度が上がります。
