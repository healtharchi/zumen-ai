🧪 これは「スマート見積＋図面AI」の実験版です

いつものスマート見積システムに、「図面を放り込むと、外周長・床面積などの入力欄にAIが自動で数字を入れてくれる」機能を実験的に足したものです。本番の公開ツールとは別物（試作）で、精度を確かめる段階です。

使う前の準備（最初に1回だけ）

  1. 見積を開いて「1. 規模」タブを表示します。
  2. 一番上の 🧪 図面AIで自動入力 パネルの中の「🔑 Claude APIキー」を開きます。
  3. お持ちの Claude API キー（sk-ant-…）を貼り付け、「🔒 キーを保存」を押します。
  4. キーはこの端末のブラウザの中だけに保存されます（外部サーバーには送りません。送り先はClaude公式のみ）。

図面から自動入力する流れ

  1. 「1. 規模」タブの 🧪 図面AIで自動入力 パネルに、図面（PDF・JPG・PNG）をドラッグ＆ドロップします。
  2. 図面の種類を選びます（平面図／立面図／屋根伏図）。種類で読み取る数値が変わります。
  3. 縮尺の手がかり（任意）に、わかっている寸法を1つ入れると精度が上がります（例：「南面の通り芯が7280mm」「1マス910mm」）。
  4. オレンジの 「🔍 図面をAIで読み取って入力欄に反映」 を押します（数十秒）。
  5. 読み取れた値が、下の 外周長・建築面積・各階床面積などの欄に自動で入ります。右側の「🔄 自動算出」と見積金額もその場で更新されます。
  6. 入った数字は、下の入力欄でいつでも手直しできます（AIはあくまで下書き）。

仕組みのポイント

  • AIが自信を持って読めた値（信頼度60%以上）だけを自動入力します。
  • 読めなかった項目は空欄のまま＝従来どおり坪数などからの係数推定で見積が成立します（破綻しません）。
  • パネルの「入力欄へ反映した値」で、何をどの信頼度で入れたか確認できます。

うまくいかないとき

  • 「APIキーが正しくないようです(401)」→ 🔑からキーを入れ直してください。
  • 何も自動入力されない→ 縮尺の手がかりに、わかる寸法を1つ入れてみてください（精度の生命線です）。
  • 図面がぼやけている→ できるだけ鮮明な画像／PDFを使うと精度が上がります。
